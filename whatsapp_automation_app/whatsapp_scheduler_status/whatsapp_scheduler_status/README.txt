# WhatsApp Automation Tools

## Scripts Included

1. `scheduled_sender.py` — Schedule messages to 10 WhatsApp groups.
2. `text_status_uploader.py` — Post a text-only status from each WhatsApp account.

## Instructions

1. Install dependencies:
   pip install -r requirements.txt

2. Make sure Chrome has 10 profiles:
   - Profile 1 through Profile 10
   - Each must be logged into a WhatsApp Web session

3. Run either script:
   python scheduled_sender.py
   python text_status_uploader.py
